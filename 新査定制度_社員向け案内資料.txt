　　　　　新しい査定・昇給のしくみについて
　　　　　　　　　　　　タイガーラック株式会社
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


　このたび、2026年より全社員を対象とした新しい査定・昇給制度を導入します。
　この資料では、制度の概要・評価のしくみ・みなさんへの影響を説明します。
　ご不明な点は、所属部署の部長、または総務にお気軽にご相談ください。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　１．なぜ新しい制度にするのか
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　これまでの評価は、基準が曖昧で「何をすれば評価されるか」が
　わかりにくい面がありました。

　今回の制度では、以下を明確にします。

　　◆ 何を・どの水準でこなせば評価されるか（評価基準の明示）
　　◆ 評価結果がどう給与に反映されるか（昇給ルールの透明化）
　　◆ どうなれば昇格できるか（等級・昇格条件の明文化）

　「頑張った人が正しく報われる仕組み」を目指します。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　２．制度の全体像
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　本制度は、以下の4つの仕組みで構成されています。

　　┌──────────────────────────────────────────────────────────┐
　　│  ① 評価制度　→　３つの軸で、年４回評価される                  │
　　│  ② 評価方式　→　部署・役職に応じた3つの方式のいずれかで評価   │
　　│                  　・KPI方式（目標数値で評価）                  │
　　│                  　・タスクログ方式（仕事の記録で評価）          │
　　│                  　・行動基準評価方式（日々の行動で評価）        │
　　│  ③ 等級制度　→　G1〜G10のグレードで役割と給与が決まる         │
　　│  ④ 昇給制度　→　12月の年次評価をもとに翌年4月から給与に反映   │
　　└──────────────────────────────────────────────────────────┘


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　３．評価のしくみ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　【評価は年４回（四半期ごと）・決算は12月】

　　第1四半期：1月〜3月 → 3月末〜4月に評価面談
　　第2四半期：4月〜6月 → 6月末〜7月に評価面談
　　第3四半期：7月〜9月 → 9月末〜10月に評価面談
　　第4四半期：10月〜12月 → 12月中旬に評価面談（年次総合評価）

　　◆ 昇給は翌年4月支給分から反映されます。
　　◆ 2026年は第2四半期（4月）から制度スタートのため、初回昇給は2027年4月です。


　【３つの評価軸】

　　すべての社員を、以下の３軸で評価します。

　　┌──────────┬───────────────────────────────────────┐
　　│  ① 役割遂行  │ あなたのポジションに期待される業務を果たせているか │
　　│  ② 成果創出  │ 数値・アウトプットなど具体的な成果を出せているか  │
　　│  ③ 組織貢献  │ チームや他部署への協力・情報共有ができているか   │
　　└──────────┴───────────────────────────────────────┘

　　◆ 各部署の特性に応じて、３軸のウェイト（比重）を調整しています。
　　　例）営業：成果創出 50% 重視 ／ 物流・総務：役割遂行 45% 重視

　　◆ 1人あたり9〜11項目を評価します。


　【評価スケール（４段階）】

　　┌───┬──────────────┬──────────────────────────────┐
　　│ S  │ 期待を大きく上回る │ 目標を大幅に超え、周囲にも好影響を与えた  │
　　│ A  │ 期待通り        │ 求められた役割を確実にこなした          │
　　│ B  │ やや不足        │ 一部において期待水準に届かなかった      │
　　│ C  │ 不足           │ 全体的に期待水準を下回った            │
　　└───┴──────────────┴──────────────────────────────┘

　　◆ 「中間」はありません。S/A/B/C の4段階です。
　　◆ 「A = 合格」です。Aが基本ラインです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　４．評価方式のしくみ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　評価の根拠をどう集めるかは、部署・役職によって3つの方式に分かれます。
　自分がどの方式かは、所属部署の部長から案内されます。

　【方式① KPI方式】　対象：営業課・受注販売課・施工／物流の管理職 など

　　毎四半期、自分で３つのKPIを設定し、部長の承認を受けます。

　　┌────────────┬──────────────────────────────────────────┐
　　│  成果KPI（数値）  │ 売上・件数・達成率など、数字で測れる目標       │
　　│  成果物KPI       │ 資料・成果物など、具体的なアウトプットの完成    │
　　│  行動KPI（改善）  │ 業務改善・再現性ある取り組みの実践            │
　　└────────────┴──────────────────────────────────────────┘

　　◆ 営業職：成果KPI ×２ ＋ 行動KPI ×１
　　◆ その他：行動KPI を中心に３つ
　　◆ KPIは本人が作成し、部長が承認します（一方的に押しつけられるものではありません）。

　【方式② タスクログ方式】　対象：クリエイティブ事業部・マーケティング企画部・事業推進部 など

　　四半期中に行ったタスクをRedmineに記録し、その内容をもとに評価します。
　　KPIのような期初目標は設定せず、日々の仕事の記録が評価根拠になります。

　　◆ タスクは種別（制作・企画・改善・運用・調査分析・支援連携）と
　　　 難易度（XL・L・M・S）を設定して記録します。
　　◆ 週に1回以上更新することで評価の信頼性が高まります。

　【方式③ 行動基準評価方式】　対象：施工・物流の現場スタッフ・総務スタッフ など

　　毎日の定型作業において「どのように仕事をしているか」を評価します。
　　目標設定や記録の提出は不要で、上司（部長・係長）の日常観察が主な根拠です。

　　◆ 安全・正確さ・手順遵守・チームへの貢献などが評価の中心です。
　　◆ 面談で評価根拠（観察した具体的な行動）を必ず説明します。

　　◆ どの方式でも、評価結果は最終的に同じ「役割遂行・成果創出・組織貢献」の
　　　 3軸スコアに集約され、統一された評価シートで総合評価が算出されます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　５．フィードバック面談
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　四半期ごとに、部長（または直属の責任者）との面談を実施します。

　　面談では必ず以下を行います：

　　　① 評価結果とその理由の説明
　　　② あなたの「強み」の言語化
　　　③ 改善点の特定と具体的なアドバイス
　　　④ 次の四半期のアクション定義

　　◆ 「なぜこの評価なのか」を必ず説明する義務が評価者にあります。
　　◆ 納得できない場合は、面談の場で確認・意見を述べることができます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　６．評価が公平かどうかチェックされます（キャリブレーション）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　部長が評価した結果は、役員と全部長が集まって横断的に確認します。

　　◆ 「部署によって評価が甘い・厳しい」という不公平をなくすための仕組みです。
　　◆ 部長の評価が不自然な場合は、上位側で補正が入ります。
　　◆ 評価が甘すぎる部長は、自身の評価を下げられる場合があります。

　→ あなたの評価は、１人の上司の判断だけで決まりません。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　７．等級制度（グレード）について
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　給与は「Gグレード（G1〜G10）」と「数値等級（1〜200）」の２層で決まります。

　【Gグレードの一覧】

　　┌──────┬──────────┬─────────────────────────────────────┐
　　│ G1   │ 一般（初級） │ 入社〜約2年。指示のもとで業務を正確に実行する     │
　　│ G2   │ 一般（中級） │ 入社2〜5年程度。自律的に業務をこなす              │
　　│ G3   │ 一般（上級） │ 専門性が高く、後輩の指導もできる                │
　　│ G4   │ 副主任      │ チームの日常をまとめ、主任・部長を補助する         │
　　│ G5   │ 主任        │ チームリーダー。業務計画・KPI管理を行う            │
　　│ G6   │ 係長        │ 係の業績・品質・他部署調整の責任者               │
　　│ G7   │ 課長        │ 課全体の戦略・業績管理・人材育成の責任者           │
　　│ G8   │ 次長        │ 部長を補佐し、部門全体の運営を支援する            │
　　│ G9   │ 部長        │ 部門の経営責任者。会社の方針を体現する             │
　　│ G10  │ 本部長      │ 複数部門を統括する最上位管理職。代表取締役が任命   │
　　└──────┴──────────┴─────────────────────────────────────┘

　【数値等級のしくみ】

　　・等級は1〜200。1ステップ上がるごとに月給が＋1,500円。
　　・毎年の評価結果に応じて、ステップ数が上がります（→ ８章参照）
　　・Gグレードが上がるほど、給与の基準額・上限額も上がります。

　【昇格のしくみ】

　　昇格（Gグレードアップ）は、以下の２つの条件が両方揃った場合に実施します：

　　　① 次のグレードに必要な「役割を発揮できている」こと
　　　② 組織上の「空きポスト」があること

　　◆ 役割を発揮していてもポストがない場合は「昇格保留」。
　　　 その場合は、数値等級の上限を引き上げるなど、別の形で報います。

　【承認ステップ】
　　　G1〜G3：部長判断
　　　G4〜G6：役員レビュー後に決定
　　　G7〜G8：役員承認
　　　G9　　：代表取締役の最終確認
　　　G10　　：代表取締役による任命（数値等級による自動昇格はなし）


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　８．昇給のしくみ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　年１回（12月末）の総合評価をもとに、翌年4月から新しい給与が適用されます。

　【評価と昇給ステップ数の目安】

　　┌──────┬─────────────────────────┬──────────────────────┐
　　│ 評価  │ 数値等級の増加（目安）           │ 月給増加額（目安）       │
　　├──────┼─────────────────────────┼──────────────────────┤
　　│  S   │ ＋6ステップ（最大8ステップ）    │ ＋9,000〜12,000円 / 月  │
　　│  A   │ ＋4ステップ（最大5ステップ）    │ ＋6,000〜 7,500円 / 月  │
　　│  B   │ ＋2ステップ（最大3ステップ）    │ ＋3,000〜 4,500円 / 月  │
　　│  C   │ ＋0ステップ（現状維持）          │ 変動なし                 │
　　└──────┴─────────────────────────┴──────────────────────┘

　　◆ 部長の裁量（評価ランク内）で最大値まで引き上げることができます。

　　◆ 評価が「C」でも給与は下がりません（現状維持）。
　　◆ 評価が「A」なら、毎年着実にステップアップします。

　【部署業績による調整】

　　昇給額は、所属部署の業績によって次のとおり調整されます：

　　　好調（目標120%以上）… ×1.2倍
　　　標準（目標80〜120%）… ×1.0倍（変動なし）
　　　不振（目標80%未満） … ×0.8倍

　　◆ 個人の評価が良くても、部署の業績が振るわない場合は調整が入ります。
　　◆ 逆に、部署が好調なら個人の昇給が手厚くなります。

　【部長裁量について】

　　部長は、同じ評価ランク内で昇給額を一定範囲で調整できます。
　　ただし、評価ランクの範囲を超えることは禁止されています。
　　（例：B評価の人にS相当の昇給はできない）

　９．よくある質問
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

　Q. 評価は部長１人が決めるのですか？
　A. 部長が一次評価を行いますが、役員・全部長による「キャリブレーション」
　　 で横断チェックが入ります。一人の判断だけで最終確定しません。

　Q. 評価に不満がある場合はどうすればいいですか？
　A. まず面談の場で評価理由の説明を求めてください。面談で解決しない場合は
　　 総務・役員に相談できる窓口を設ける予定です。

　Q. C評価になると給与は下がりますか？
　A. 下がりません。C評価は「昇給なし（現状維持）」です。

　Q. 自分の評価方式はどうやって決まりますか？
　A. 部署・役職によってあらかじめ決まっています。制度開始時に部長から案内されます。
　　 KPI方式の方は四半期初に目標を自分で作成し、部長が承認します。
　　 タスクログ方式の方はRedmineで日常的に仕事を記録します。
　　 行動基準方式の方は記録・目標設定は不要で、日々の行動を上司が観察します。

　Q. 昇格できる人数に制限はありますか？
　A. 昇格は「役割を発揮しているか」と「ポストがあるか」で決まります。
　　 単純なポイント争いではなく、能力・役割充足の基準です。

　Q. 今の等級（Gグレード）はいつ通知されますか？
　A. 制度開始時に、個別に書面でお知らせします。
